Typical IDE features

  • Built-in terminals and debuggers.
  • Git integration and extensions marketplace.
  • Tasks, formatters, and linters.

Why VS Code here?

Choosing an IDE

If you already have a favorite editor, you can keep using it as long as you can follow course directions. For consistency, we recommend VS Code.

Considerations

  • OS support: Windows, macOS, Linux.
  • Performance on your machine.
  • Language & framework support.
  • Ecosystem/plugins and community.
  • Team/program standards.

Our standard

  • Visual Studio Code (Microsoft) ‚Äî free and popular.
  • Great for web: HTML, CSS, JS/TS, React.
  • Shared tips and screenshots use VS Code.

Install and set up Visual Studio Code
