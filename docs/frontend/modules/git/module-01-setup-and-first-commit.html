Set up Git (10‚Äì20 min)

  • Install & configure Git (name, email, default branch, helpful defaults). Prefer not to use the terminal? Install GitHub Desktop. Docs
  • Handle credentials the right way (GitHub CLI or Git Credential Manager) and avoid repeated password prompts. Docs
