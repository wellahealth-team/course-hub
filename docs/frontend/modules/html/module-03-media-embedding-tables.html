Troubleshooting

Media won‚Äôt load

  • Check file paths; store assets relative to the HTML file.
  • Always provide multiple sources for audio/video (`.mp4`, `.webm`, `.mp3`).
  • Set `controls` and `preload="metadata"` to improve UX.

Table structure feels confusing

  • Sketch the table first‚Äîidentify row groups, headers, and spanning cells.
  • Use the W3C validator to catch colspan/rowspan mismatches.
  • Use `scope="rowgroup"`/`scope="colgroup"` or `headers` attributes for clarity.
