Outcomes

  • Explain what HTML is for; distinguish elements vs. attributes; know block vs. inline and nesting rules.
  • Create a minimal, valid HTML document and open it in a browser.
  • Use <title>, <meta charset>, <meta name="viewport">, <meta name="description">; optionally link a favicon.
  • Mark up text with headings, paragraphs, lists, emphasis, quotes, and inline code.
  • Create accessible hyperlinks: absolute, relative, fragment links, and email/telephone links; use target/rel safely.
