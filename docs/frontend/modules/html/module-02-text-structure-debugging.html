Troubleshooting

Text semantics look wrong

  • Use the Accessibility pane in DevTools to check computed roles for <abbr>, <mark>, and citations.
  • Ensure figures include <figcaption> so screen readers announce the context.
  • Remember that <b> and <i> are visual only‚Äîswap for <strong>/<em> when meaning matters.

Layout feels disorganized

  • Sketch the page outline first; each heading level should introduce a new idea, not styling.
  • Check for unopened or unclosed tags‚Äîthe validator link above flags line numbers.
  • Group related content inside <section> with a heading; move side notes into <aside>.
